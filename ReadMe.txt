XML Editor v0.5
===============

Introduction
------------

This is a simple XML editor. It currently only provides support for viewing XML
files. It was largely written as a test harness for my XML library.

Releases
--------

Stable releases are formally packaged and made available from my Win32 tools page:
http://www.chrisoldwood.com/win32.htm

The latest code is available from my GitHub repo:
https://github.com/chrisoldwood/XMLEdit

Installation
---------------

Run setup.exe.

or

Copy the files XMLEdit.* to a folder.

Uninstallation
-----------------

Delete the files listed above.
